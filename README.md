# 安安的学习乐园 (Anan's Learning Park) 🎡

这是一个专为小学生（特别是低年级小朋友）设计的趣味互动学习网页平台。灵感来源于色彩鲜艳的日系海报设计，旨在通过游戏化、具象化的方式帮助孩子理解抽象的数学概念。

## ✨ 项目亮点

*   **童趣视觉设计**：
    *   采用日系海报配色（天空蓝、草地绿、柠檬黄），清新活泼。
    *   包含节日彩旗、缤纷彩屑、手绘风边框等细节，营造游园会氛围。
*   **沉浸式体验**：
    *   **全屏 PWA 设计**：完美适配 iPad/平板设备，支持“添加到主屏幕”运行，无地址栏干扰。
    *   **动态布局**：使用 `100dvh` 和安全区 (`safe-area`) 适配，解决移动端浏览器地址栏遮挡问题。
    *   **个性化姓名**：网页标题和欢迎语会根据小朋友的名字动态变化（如“安安的学习乐园”）。
    *   **拟真音效**：所有游戏均包含点击、投币、成功欢呼等 Web Audio 合成音效。

## 🎮 核心游戏

### 0. 🍬 糖果跳跳口算台
*   **知识点**：20 以内加减法、数轴正向/逆向跳步。
*   **玩法**：每轮 20 道随机算式，输入答案即可记录；可随时换题重刷。
*   **特色**：
    *   **胜利记录册**：右侧实时记录 20 道题的算式、答案与对错，复习时一目了然。
    *   **烟花彩纸**：每答对 1 题立刻放烟花，保持成就感。
    *   **触控数字条**：算式下方 0-20 触屏按钮，iPad 上点一点就能输入答案。
    *   **奖杯激励**：≥15 题铜杯、≥17 题银杯、全对金杯，孩子可以冲击更高目标。

### 1. ⚖️ 森林魔法集市 (Balance Scale)
    *   **知识点**：等量代换（代数思维启蒙）。
    *   **玩法**：通过拖拽物品（蘑菇、黄瓜、动物等）到天平上，观察平衡关系，解开换算谜题。
    *   **特色**：
        *   **物理天平反馈**：答错时天平会左右倾斜，直观感受数量差异。
    *   **无限随机关卡**：海量 Emoji 物品库，每次题目都不一样。
    *   **可视化规则**：用图形直观展示 $1\text{🍄} = 2\text{🥒}$。

### 2. 🪙 动物售货机 (Vending Machine)
*   **知识点**：认识人民币（元、角换算）与加法凑整。
*   **玩法**：观察商品价格，从钱包中拖拽正确的硬币（1元、5角、1角）投入售货机。
*   **特色**：
    *   **真实钱包扣款**：投币时钱包余额实时减少，建立“花钱”概念。
    *   **拟真投币体验**：硬币入孔会有清脆的金属撞击音效（双频拍频模拟颤音）。
    *   **防洗钱机制**：利用 Stack 记录投币历史，退款时原样退回刚才投入的硬币组合，逻辑严谨。
    *   **购物战利品**：买到的商品会飞入“我的包包”中展示，满足收集欲。

### 3. 🎒 书包整理大作战
*   **知识点**：根据课程安排规划物品，培养责任感与分类整理习惯。
*   **玩法**：查看明日课表和特殊提醒，点击把对应书本、水杯、口罩及额外需求装入书包，不可缺漏也不可多带。
*   **特色**：随机生成周课表与额外小要求，检查会提示缺漏或多余，让孩子体验“刚刚好”的成就感。

### 4. 🐾 动物英语接龙
*   **知识点**：一年级英语口语启蒙（问候、礼貌用语、基础句型，身体部位指令）。
*   **玩法**：看字幕从四个回应中选择正确答案；在“Touch my face”场景点对动物的鼻子、眼睛、耳朵等获取奖励星。
*   **特色**：纯前端字幕练习，无音频依赖，正误即时反馈，触摸互动降低开口恐惧。

## 📂 文件结构

*   `index.html` - **主页入口**：包含侧边栏导航、个性化设置。内置 iOS 链接防跳转脚本。
*   `math_jump.html` - **糖果跳跳口算台**：20 题答题记录、烟花和彩杯激励的加减闯关。
*   `balance.html` - **森林魔法集市**：天平平衡小游戏。
*   `vending.html` - **动物售货机**：模拟购物与找零小游戏。
*   `bag.html` - **书包整理大作战**：根据课表选择书本与必备品。
*   `english_fun.html` - **动物英语接龙**：字幕选择与触摸部位练习。
*   `README.md` - 项目说明文档。

## 📱 最佳体验指南 (iPad/平板)

本项目专为触摸屏优化，强烈建议按照以下步骤在 iPad 上运行，以获得**原生 App 般的全屏体验**：

1.  **使用 Safari 打开**：
    *   在 iPad 上使用 Safari 浏览器访问部署后的网址。
2.  **添加到主屏幕**：
    *   点击浏览器右上角的 **“分享”按钮** (⬆️)。
    *   在菜单中向下滑动，点击 **“添加到主屏幕” (Add to Home Screen)**。
    *   点击右上角的“添加”。
3.  **从桌面启动**：
    *   此时桌面上会出现一个“学习乐园”的图标。
    *   **点击图标启动**：网页将以全屏模式运行，隐藏浏览器地址栏和底部工具栏。
    *   *注：项目已内置防跳转代码，点击内部链接将保持在全屏模式，不会跳回浏览器。*
