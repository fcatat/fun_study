# 安安的学习乐园 (Anan's Learning Park) 🎡

这是一个专为小学生（特别是低年级小朋友）设计的趣味互动学习网页平台。灵感来源于色彩鲜艳的日系海报设计，旨在通过游戏化、具象化的方式帮助孩子理解抽象的数学概念。

## ✨ 项目亮点

*   **童趣视觉设计**：
    *   采用日系海报配色（天空蓝、草地绿、柠檬黄），清新活泼。
    *   包含节日彩旗、缤纷彩屑、手绘风边框等细节，营造游园会氛围。
    *   全屏无干扰设计，适配 **iPad / 平板设备** 触摸操作。
*   **沉浸式体验**：
    *   **个性化姓名**：网页标题和欢迎语会根据小朋友的名字动态变化（如“安安的学习乐园”）。
    *   **拟真音效**：所有游戏均包含点击、投币、成功欢呼等 Web Audio 合成音效。
    *   **年级体系**：内置 1-6 年级导航，方便未来扩展。

## 🎮 核心游戏

### 1. ⚖️ 森林魔法集市 (Balance Scale)
*   **知识点**：等量代换（代数思维启蒙）。
*   **玩法**：通过拖拽物品（蘑菇、黄瓜、动物等）到天平上，观察平衡关系，解开换算谜题。
*   **特色**：
    *   **物理天平反馈**：答错时天平会左右倾斜，直观感受数量差异。
    *   **无限随机关卡**：海量 Emoji 物品库，每次题目都不一样。
    *   **可视化规则**：用图形直观展示 $1\text{🍄} = 2\text{🥒}$。

### 2. 🪙 动物售货机 (Vending Machine)
*   **知识点**：认识人民币（元、角换算）与加法凑整。
*   **玩法**：观察商品价格，从钱包中拖拽正确的硬币（1元、5角、1角）投入售货机。
*   **特色**：
    *   **真实钱包扣款**：投币时钱包余额实时减少，建立“花钱”概念。
    *   **拟真投币体验**：硬币入孔会有清脆的金属撞击音效。
    *   **防洗钱机制**：投币过多退款时，会原样退回刚才投入的硬币组合，逻辑严谨。
    *   **购物战利品**：买到的商品会飞入“我的包包”中展示，满足收集欲。

## 📂 文件结构

*   `index.html` - **主页入口**：包含侧边栏导航、个性化设置和功能卡片列表。
*   `balance.html` - **森林魔法集市**：天平平衡小游戏。
*   `vending.html` - **动物售货机**：模拟购物与找零小游戏。
*   `README.md` - 项目说明文档。

## 🚀 快速开始

1.  **直接运行**：
    *   不需要任何服务器环境（纯静态 HTML/JS）。
    *   直接双击打开 `index.html` 文件即可在浏览器中运行。
2.  **iPad 最佳体验**：
    *   推荐使用 **iPad** Safari 浏览器访问。
    *   点击“分享” -> **“添加到主屏幕”**。
    *   网页将以 **全屏 App 模式** 运行，隐藏地址栏，体验最佳。
