# 安安的学习乐园 (Anan's Learning Park) 🎡

这是一个专为小学生（特别是低年级小朋友）设计的趣味互动学习网页平台。灵感来源于色彩鲜艳的日系海报设计，旨在通过游戏化、具象化的方式帮助孩子理解抽象的数学概念。

## ✨ 项目亮点

*   **童趣视觉设计**：
    *   采用日系海报配色（天空蓝、草地绿、柠檬黄），清新活泼。
    *   包含节日彩旗、缤纷彩屑、手绘风边框等细节，营造游园会氛围。
    *   全屏无干扰设计，适配 iPad 等平板设备操作。
*   **沉浸式体验**：
    *   个性化姓名设置，让每一位小朋友都有专属的主页欢迎语。
    *   内置“年级”导航体系，方便未来扩展更多内容。
*   **核心游戏：森林魔法集市 (Math Market)**：
    *   **知识点**：等量代换（代数思维启蒙）。
    *   **玩法**：通过拖拽物品（蘑菇、黄瓜、动物等）到天平上，观察平衡关系，解开换算谜题。
    *   **特色**：
        *   **物理天平反馈**：答错时天平会左右倾斜，直观感受数量差异。
        *   **无限随机关卡**：内置海量 Emoji 物品库，每次题目都不一样，常玩常新。
        *   **可视化规则**：用图形展示 $1\text{🍄} = 2\text{🥒}$，无需阅读枯燥文字。

## 📂 文件结构

*   `index.html` - **主页入口**：包含侧边栏导航、个性化设置和功能卡片列表。
*   `math_game.html` - **森林魔法集市**：代数思维训练小游戏。
*   `README.md` - 项目说明文档。

## 🚀 快速开始

1.  **直接运行**：
    *   不需要任何服务器环境。
    *   直接双击打开 `index.html` 文件即可在浏览器中运行。
2.  **最佳体验**：
    *   推荐使用 **iPad / 平板电脑**。
    *   在 Safari 浏览器中点击“分享” -> “添加到主屏幕”，即可获得全屏 App 般的沉浸体验。
